# IRIS
 
### Toggle Gestures
Right Hand Fist

## Controls (Right Hand)

### Left Click
Thumb touching Index finger

### Right Click
Thumb touching Middle finger

### Enter Key
Thumb touching Ring finger

### Voice Recording
Index + Middle + Ring fingers up

### Arrow Keys

## Left Arrow
Thumb pointing left (all other fingers down)

## Left Hand Gestures

### Scroll Up
Index + Middle fingers

### Scroll Down
Only Index finger

### Drag
Thumb touching Index

### Alt-Tab Window Switcher
Left hand fist

 Action:
 Opens - Alt-Tab
 Hold - cycles through windows
 Release - selects window

### Backspace
Thumb touching Pinky\*\*

## Virtual Keyboard

### Toggle Virtual Keyboard
Peace sign on BOTH hands
Hold for 0.5 seconds

## Virtual Keyboard Controls

### Hover
Index finger

### Key Press

Index + Thumb up
